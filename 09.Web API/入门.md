## 什么是REST？

WebAPI的风格：

1、**面向过程**

RPC：控制器/操作方法的形式来调用接口。把HTTP当成传输数据的通道，不关心HTTP谓词。

通过QueryString，请求报文体向服务器传递数据。

2、**面向REST**

按照HTTP语义来使用HTTP协议。

- HTTP谓词：

GET，POST，PUT（整体更新），DELETE，PATCH（局部更新）

- URL用于资源的定位

- 什么是幂等？

  这个动作发生一次和发生n次的结果是一样的。
